# Auto Lyrics Furigana (自动歌词注音工具)

一个基于 Python 的自动化脚本，用于批量处理本地音乐文件的内嵌歌词。它会自动识别歌词中的日文汉字，并利用 `pykakasi` 库在汉字后添加平假名注音（Furigana）。

适用于 MP3，wav 和 FLAC 格式，支持递归扫描文件夹。

## ✨ 功能特点

- **自动识别日文**：通过正则判断，仅对包含汉字和假名的日文行进行处理，**不会误伤中文翻译**或英文歌词。
- **智能防重复**：如果歌词中已经存在注音（如 `未来(みらい)`），脚本会自动跳过，防止出现双重括号。
- **格式支持**：
  - ✅ **MP3** (ID3v2 USLT 标签)
  - ✅ **FLAC** (Vorbis LYRICS/UNSYNCEDLYRICS 标签)
  - ✅ **WAV** (仅支持带 ID3 标签的 WAV)
- **安全扫描**：自动跳过非音频文件（如 .jpg, .nfo, .cue），仅处理目标文件。

## 🛠️ 依赖库

在运行脚本之前，请确保安装了 Python 3 以及以下第三方库：


pip install mutagen pykakasi

# 注意事项
## 备份数据：虽然脚本经过测试，但涉及批量修改文件标签，强烈建议在运行前备份你的音乐文件。
## 注音准确性：注音基于 pykakasi 词典，绝大多数情况下是标准的训读/音读。对于歌曲中特殊的“熟字训”或歌手的特殊读法（如 宇宙 读作 そら），脚本无法识别，可能需要人工校对。
